Online Eye Care System

The Online Eye Care System is a web-based application developed using ASP.NET and SQL Server to manage eye care services digitally. It allows patients to book appointments, store prescriptions, and order eye care products online.

Features

Patient registration and login

Online appointment booking

Digital patient record management

Browse and order eye care products

Admin management of doctors, patients, products, and services

Report generation

Technologies Used

Frontend: HTML, CSS, ASP.NET, C#

Backend: SQL Server

Framework: .NET Framework 4.0

IDE: Microsoft Visual Studio 2015
